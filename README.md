# Matheesha's Hackpad Controller
Macropad used to control a robotic arm.

## Inspiration
I wanted to control a robotic arm I'm working on using buttons for manual testing, so I decided to make this hackpad.

- 9x Cherry MX Switches  
- 2x SK6812 MINI LEDs  
- 1x XIAO RP2040  
- 9x Blank DSA Keycaps  
- 4x M3x16 Bolts  
- 4x M3 Heatset Inserts
